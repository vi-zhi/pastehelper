# PasteHelper - 智能粘贴助手

## 项目简介

PasteHelper 是一款基于 Java Swing 开发的桌面应用程序，旨在解决日常工作中频繁复制粘贴相同文本内容的效率问题。通过自定义快捷键绑定常用文本片段，实现一键快速粘贴，大幅提升工作效率。

## 解决的问题

### 1. **重复文本输入效率低下**
在日常开发、文档编写、客服回复等场景中，经常需要反复输入相同的文本内容（如邮箱地址、电话号码、标准回复模板、代码片段等）。传统方式需要每次都手动复制或重新输入，耗时且容易出错。

### 2. **剪贴板管理混乱**
系统剪贴板只能保存最后一次复制的内容，当需要在多个常用文本之间切换时，需要反复复制，操作繁琐且容易混淆。

### 3. **缺乏快捷访问机制**
没有便捷的方式为常用文本设置快捷键，无法在任意应用中快速调用预设的文本内容。

## 核心功能

### ✨ 主要特性

- **全局快捷键监听**：使用 jnativehook 实现系统级键盘监听，在任何应用中都能响应快捷键
- **自定义文本模板**：可添加、编辑、删除常用的文本片段
- **灵活快捷键绑定**：支持 `Ctrl+Shift+数字键` 和 `Ctrl+Alt+数字键` 组合快捷键
- **可视化配置界面**：提供友好的 Swing GUI 界面管理粘贴项
- **持久化存储**：配置自动保存到 JSON 文件，重启后依然有效
- **智能粘贴执行**：按键释放后自动执行粘贴，避免误触发

### 🎯 使用场景

- 💻 **开发人员**：快速粘贴常用代码片段、API 端点、配置信息
- 📝 **文档编写**：快速插入标准段落、引用文本、格式化模板
- 📧 **客服工作**：快速回复常见问题、标准话术
- 📊 **数据录入**：快速填写重复性字段信息

## 技术栈

- **Java 8** - 核心编程语言
- **Spring Boot 2.7.14** - 应用框架和依赖注入
- **Java Swing** - 图形用户界面
- **jnativehook 2.2.2** - 全局键盘钩子库
- **Gson 2.11.0** - JSON 序列化/反序列化
- **Lombok 1.18.46** - 简化 Java 代码
- **Maven** - 项目构建和依赖管理
